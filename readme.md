# 実践的プログラミング　

## LexicalAnalyzer (newlang3)
字句解析プログラム
    - LexicalAnalyzer   
        字句解析インターフェース
    - LexicalAnalyzerImpl   
        字句解析の実行クラス
    - LexicalType  
        字句の型
    - LexicalUnit
        １字句分のクラス
    - Main
    - Value
        １字句の値のインターフェース
    - ValueImpl
        字句値の実行クラス
    - ValueType
        値の型
